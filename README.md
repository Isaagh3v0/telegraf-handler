```

```
## – Compile And Start
### 1. Install dependencies
```bash
yarn add
```

### 2. Renaming config file
`Rename config file .env.example to .env.`

### 3. Adding token
`Replace TOKEN="THERETOKEN" field in the .env configuration file.`

### 4. Building project
```bash
yarn build
```

### 5. Start bot
```bashs
yarn start
```
